Что такое Агрегат?

Это как Cortana под Windows 10, только лучше! Потому что Агрегат:


  1. Понимает русский язык
  2. Работает на компьютере с любой ОС
  3. Запускается даже на Raspberry Pi
  4. Умеет управлять любой удаленной техникой
  5. Гибко настраивается под Ваши требования
  6. Имеет открытый код и API для разработчиков

Зачем он мне?

Управлять техникой с помощью своего голоса - это действительно удобно! Вместо того, чтобы набирать сообщение или запрос на клавиатуре, его можно произнести. Чтобы открыть сайт - достаточно сказать его адрес. Поставить таймер на смартфоне или создать напоминание? Легко - не отрываясь от работы на ПК!

Агрегат - словно конструктор Lego

Вы сами выбираете, какие модули установить и какими функциями пользоваться

Все зависит лишь от Ваших настроек. Например, Вы можете использовать любой голос для Вашего ассистента. Или заменить распознавание речи Google на сервис от Яндекса. Можно использовать клавиатуру для запуска ассистента, а можно - датчик приближения или голос. Установите или разработайте нужный модуль - и соберите свой Агрегат!

Удаленное управление

Включив в мобильном ассистенте Дуся функцию удаленного управления, Вы можете отдавать ему голосовые команды прямо с вашего компьютера! При этом ответ вы услышите не из смартфона, а из динамиков своего ПК.


Даже не нужно брать телефон в руки! Вы можете запускать любые скрипты удаленно и получать результат прямо на экране Вашего компьютера.


"Поставь песню про зайцев", "Напомни покормить кота через 2 часа", "Какие планы на вечер", "Создай заметку" - вот примеры того, как вы можете командовать своему смартфону, не отрываясь от работы на ПК.


Агрегат работает как единая точка управления любой техникой. И при этом - это не штатная функция самого Агрегата, а лишь один из модулей.

Raspberry Pi

Агрегат можно запустить даже на микрокомпьютерах типа Raspberry Pi. Нет ничего лучше для создания компактного голосового помощника, который поместится в небольшой коробочке. К нему можно подключить разнообразные сенсоры и превратить в настоящего робота.

Открытый код

Создавать голосового ассистента - легко!

Больше функций - больше возможностей. Только открытый проект развивается быстро и дает настоящую свободу воображению.


Агрегат - это проект с открытым исходным кодом. С помощью открытого API Вы можете сами создавать дополнительные функциональные модули и публиковать их.


Исходные коды модулей и документация для сторонних разработчиков доступны в репозитории GitHub.
